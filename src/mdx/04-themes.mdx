# Themes

<div style={{ 
  marginTop: '2rem',
  marginBottom: '2rem',
  padding: '2rem',
  backgroundColor: 'var(--bg-secondary, #f8f9fa)',

There are three default themes in archUI, each with unique characteristics. These themes are inspired by Ancient Greek architectural orders: **Ionic**, **Doric**, and **Corinthian**. Each theme offers a distinct visual identity and design language to match your project's aesthetic needs.

</div>

## Ionic

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',
  padding: '1.5rem',
  backgroundColor: 'var(--bg-accent-light, #e0e7ff)',
  borderRadius: '8px',
  borderLeft: '4px solid var(--bg-accent, #3b82f6)'
}}>

This is the `default` theme, and requires no additional configuration to get started. Inspired by the volutes and flowing curves of ancient Greek architecture, Ionic UI creates a welcoming and visually engaging user experience.

</div>

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',
  padding: '1.5rem',
  backgroundColor: 'var(--bg-secondary, #f8f9fa)',
  borderRadius: '8px'
}}>

With its approachable aesthetic, this theme harmonizes beautifully with:

- **Calm** brands
- **Innovative** brands
- **Warm & inviting** brands
- **Playful** brands
- **Beauty** brands

</div>

### Characteristics

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',
  padding: '1.5rem',
  backgroundColor: 'var(--bg-secondary, #f8f9fa)',
  borderRadius: '8px'
}}>

The Ionic UI theme is defined by these distinctive characteristics:

- **Rounded corners** that soften visual edges
- **Generous paddings** for comfortable spacing
- **Borderless design** for seamless integration
- **Subtle decorative touches**
- **Soft, blurred shadows** that add depth
- **An overall pleasant, graceful appearance**

</div>

### Characteristics

The Doric UI theme is distinguished by the following key characteristics:

- **Sharp edges**
- **Contrasting border colors**
- **Lack of aesthetic details**
- **Subtle shadows**
- **Shorter and sturdier appearance**

</div>

## Corinthian

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',

The most elaborate and decorative of the three themes, Corinthian UI embodies **sophistication and refined elegance**. Drawing inspiration from the ornate acanthus leaves and intricate details of ancient Greek columns, this theme creates a luxurious and polished user experience.

</div>

### Characteristics

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',

The Corinthian UI theme is distinguished by the following key characteristics:

- **Subtle rounded corners** for refined edges
- **Delicate padding** that maintains spaciousness
- **Light-gray borders** for elegant definition
- **Rich ornamental details** throughout
- **Soft, layered shadows** for dimensional depth
- **Gradient backgrounds** that add visual sophistication
- **Slender, svelte appearance** that conveys grace

</div>
