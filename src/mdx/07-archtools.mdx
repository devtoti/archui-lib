# ArchTools

<div style={{ 
  marginTop: '2rem',
  marginBottom: '2rem',

**ArchTools** is a React-based web application that demonstrates ArchUI in a real-world context. Built specifically for architects and designers, ArchTools provides a suite of practical widgets and tools that leverage the ArchUI design system to create a cohesive, professional user experience.

</div>

<div style={{ 
  marginTop: '1.5rem',

## Available Tools

### Scale Calculator

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',
  padding: '1.5rem',
  backgroundColor: 'var(--bg-secondary, #f8f9fa)',
  borderRadius: '8px'
}}>

The **Scale Calculator** helps architects and designers convert measurements between different scales, making it easy to work with drawings and plans at various scales. This tool demonstrates archUI's form components and calculation interfaces.

</div>

### Plan Maker

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',
  padding: '1.5rem',
  backgroundColor: 'var(--bg-secondary, #f8f9fa)',
  borderRadius: '8px'
}}>

The **Plan Maker** provides an interactive interface for creating and editing architectural plans. This tool showcases archUI's layout components, drawing capabilities, and complex interface patterns.

</div>

### SVG to DWG Converter

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',

The **SVG to DWG Converter** allows users to convert vector graphics from SVG format to DWG (AutoCAD Drawing) format. This utility demonstrates archUI's file handling interfaces and conversion workflows.

</div>

### Staircase Creation Playground

<div style={{ 
  marginTop: '1.5rem',

The **Staircase Creation Playground** is an interactive tool for designing and visualizing staircases with various parameters. This playground demonstrates archUI's interactive components, 3D visualization capabilities, and complex parameter controls.

</div>

## Design System Integration

<div style={{ 
  marginTop: '1.5rem',
  marginBottom: '1.5rem',
  padding: '1.5rem',
  backgroundColor: 'var(--bg-accent-light, #e0e7ff)',

ArchTools serves as a **living example** of ArchUI in action, demonstrating:

- **Theme flexibility**: How different themes (Doric, Ionic, Corinthian) can be applied across a complex application
- **Component composition**: How ArchUI components work together to build sophisticated interfaces
- **Real-world patterns**: Practical examples of common UI patterns using ArchUI components
- **Accessibility**: How WCAG-compliant components create inclusive experiences
- **Performance**: How the tokenized design system enables fast, efficient applications

</div>
