## 00 GETTING Started

### Installation

```bash
npm install @devtoti/archui-lib
```

### Basic Usage

```tsx
import { Button } from "@devtoti/archui-lib";
import "@devtoti/archui-lib/styles";

function App() {
  return (
    <Button variant="primary" size="md">
      Click me
    </Button>
  );
}
```

### Using Themes

Wrap your application with a `ThemeProvider` to apply one of the three themes:

```tsx
import { ThemeProvider } from "@devtoti/archui-lib";

function App() {
  return (
    <ThemeProvider theme="ionic" setTheme={() => {}}>
      {/* Your app components */}
    </ThemeProvider>
  );
}
```

## Design Philosophy

archUI is inspired by classical architecture principles:

- **Proportion**: Balanced spacing and sizing
- **Harmony**: Consistent design language across components
- **Clarity**: Clear visual hierarchy and communication
- **Elegance**: Refined aesthetics that stand the test of time
